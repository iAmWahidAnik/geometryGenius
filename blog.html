Q#01: What is DOM and what is the purpose of DOM?

Dom is programing interface for web documents. DOM represents the page for change the document style, structure and content. DOM represents the document as nodes and objects, that can, programming languages can interact with the page.

Q#02: How will you select HTML elements using DOM? Name the DOM methods.

we can select html elements using DOM Selectors. The DOM selectors are :

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

Q#03: What is event bubble?

Event bubbling in DOM. It happens when an element recive an event and the event bubbles propagated ot its parent elemnt and parents parent element in the DOM tree untill it get into the root element.

Q#04: What is a callback function and why will you use it?

A JavaScript callback is a function which is to be executed after another function has finished execution.

Callback functions are often used to continue code execution after an asynchronous operation has completed.
